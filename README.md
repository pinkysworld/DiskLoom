# DiskLoom Website

Static product website for DiskLoom, deployed to GitHub Pages from `main`.

## Gumroad launch

DiskLoom is published at `https://michelpicker.gumroad.com/l/diskloom` with a
signed and notarized DMG, a 14-day in-app trial, current screenshots, and a
EUR 39 one-time license for up to three Mac activations. The launch campaign
offers the first 100 licenses for EUR 29 through the `LAUNCH29` offer code.

The `learn/` directory contains public drive and backup health guides. Reusable
Product Hunt, affiliate, and social launch copy lives in `marketing/`. Run
`scripts/build-marketing-videos.sh` to rebuild the five short MP4 demos from
the current application screenshots.

## Local preview

Open `index.html` directly or serve this directory with any static web server.
