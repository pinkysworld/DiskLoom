## 3. External USB truth

Why can Windows read an external drive's SMART data while macOS cannot?

Often the USB bridge or macOS transport blocks command passthrough. DiskLoom tests supported paths and reports the result honestly: unknown is never turned into healthy.

https://minh.systems/DiskLoom/learn/usb-smart-macos.html
