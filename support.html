Support

USB and external drives

External SMART access depends on the enclosure bridge and the command path exposed by macOS. A drive can mount normally while SMART passthrough remains unavailable. DiskLoom reports this as a compatibility limitation, not a drive failure.

Support report

In DiskLoom, use Export to create a polished PDF health report, a privacy-redacted support report, or a CSV history file. Review any report before sharing it. Serial numbers are redacted by default in support exports.

Diagnostic engine

DiskLoom includes a universal Apple Silicon and Intel build of smartctl. Homebrew and Rosetta are not required. A connection can still block SMART passthrough even when the diagnostic engine is available.

Lifetime and speed results

SSD lifetime projections become more useful as local history grows. HDD lifetime figures are age-based heuristics, not manufacturer guarantees. The optional speed test reads an existing accessible file for a few seconds and does not create test data on the drive.

License activations

The Personal license supports three Gumroad activations. Remove the local key in DiskLoom Settings before transferring the app. If a lost or replaced Mac exhausts the activation count, contact support for a Gumroad reset.

Contact

For now, use GitHub Issues and include the DiskLoom version, macOS version, and a redacted support report. Do not post license keys or unredacted serial numbers.
