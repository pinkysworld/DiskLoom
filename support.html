Support

USB and external drives

External SMART access depends on the enclosure bridge and the command path exposed by macOS. A drive can mount normally while SMART passthrough remains unavailable. DiskLoom reports this as a compatibility limitation, not a drive failure.

Support report

In DiskLoom, choose Monitor → Export Support Report. Review the report before sharing it. Serial numbers are redacted by default.

License activations

The planned Personal license supports three active Macs. Purchase and license support will be available when DiskLoom launches on Gumroad.

Contact

For now, use GitHub Issues and include the DiskLoom version, macOS version, and a redacted support report. Do not post license keys or unredacted serial numbers.
